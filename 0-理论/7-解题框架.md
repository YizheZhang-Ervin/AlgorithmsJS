# 回溯与分支限界  
适用:求解搜索问题和优化问题   
搜索空间: 树,结点对应部分分解向量,树叶对于可行解  
搜索过程: 系统的方法隐含遍历搜索树  
搜索策略: 深度优先,宽度优先,函数优先,宽深结合  
结点分支判断条件:满足约束条件(分支扩张解向量)/不满足约束条件:回溯该节点的父节点  
结点状态:动态生成(白:未探索,灰:已探索未访问,黑:已访问)  
存储: 当前路径
